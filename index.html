Banco total desde cero — Entorno Estratégico de TI

Hecho para enseñar primero qué significa cada cosa y después cómo se aplica. La idea es que alguien que no ha estudiado todavía pueda empezar por la guía, luego hacer conceptos en súper fácil y subir poco a poco.
Recomendación de uso: primero abre la guía y repasa por bloques. Después haz Conceptos → Súper fácil. Cuando eso ya salga mejor, pasa a Conceptos → Fácil. Ya luego subes a medio, difícil y a la ruta práctica.

Guía resumida para entender todo antes del examen

1. Base para empezar desde cero
Antes de meterte a estrategias o diagnósticos, primero debes entender las palabras base. Aquí la idea es simple: una organización usa recursos, conocimientos, herramientas y habilidades para resolver problemas. Cuando todo eso se ordena, puede planear mejor lo que hará con la tecnología.
Proceso
Palabra clave para memorizar: secuencia
Explicación: Serie ordenada de pasos para llegar a un resultado.
Iterativo
Palabra clave para memorizar: repetición
Explicación: Algo que se repite para corregir, ajustar o mejorar.
Proceso iterativo
Palabra clave para memorizar: ciclo
Explicación: Conjunto de pasos que se repite por ciclos para mejorar.
Estrategia
Palabra clave para memorizar: camino
Explicación: Plan general para alcanzar un objetivo.
Estrategia tecnológica
Palabra clave para memorizar: alineación
Explicación: Plan para usar la tecnología a favor de los objetivos de la organización.
Entorno
Palabra clave para memorizar: alrededor
Explicación: Todo lo que rodea e influye en una persona, empresa o sistema.
Entorno empresarial
Palabra clave para memorizar: contexto
Explicación: Todo lo que rodea a una empresa y afecta su funcionamiento.
Capacidades de TI
Palabra clave para memorizar: capacidad
Explicación: Recursos, conocimientos, herramientas y habilidades tecnológicas con que cuenta una organización.
Tecnología
Palabra clave para memorizar: solución
Explicación: Aplicación de conocimientos, herramientas, técnicas y métodos para resolver problemas o facilitar tareas.
Plan
Palabra clave para memorizar: orden
Explicación: Organización previa de qué se hará, cómo, con qué y para qué.
Recursos

Conocimientos

Herramientas

Habilidades tecnológicas
Palabra clave para memorizar: práctica

Empresa

Organización

Técnica

Método

Idea

2. Proceso de la planeación estratégica de TI
La planeación estratégica de TI no es hacer cosas al azar. Primero se analiza la situación, luego se formula el plan, después se implementa y al final se evalúa. Como el entorno cambia, el proceso vuelve a repetirse para mejorar.
Análisis
Palabra clave para memorizar: diagnóstico
Explicación: Etapa en la que se entiende la situación actual antes de actuar.
Formulación
Palabra clave para memorizar: diseño
Explicación: Etapa en la que se decide la estrategia o plan a seguir.
Implementación
Palabra clave para memorizar: ejecución
Explicación: Etapa en la que el plan se pone en práctica.
Evaluación
Palabra clave para memorizar: revisión
Explicación: Etapa en la que se revisa si lo hecho funcionó y qué debe corregirse.
Planeación estratégica de TI
Palabra clave para memorizar: alineación
Explicación: Decidir cómo usar la tecnología para ayudar a cumplir objetivos organizacionales.
3. Jerarquía de la estrategia
La pirámide estratégica va de lo más general a lo más concreto. La visión dice a dónde se quiere llegar; la misión explica para qué existe la organización; los objetivos marcan grandes resultados; y las metas bajan eso a algo medible y con fecha.
Visión
Palabra clave para memorizar: futuro
Explicación: Imagen futura deseada de la organización.
Misión
Palabra clave para memorizar: propósito
Explicación: Razón de ser fundamental de la organización.
Objetivo estratégico
Palabra clave para memorizar: resultado
Explicación: Gran resultado que se busca lograr a largo plazo.
Meta
Palabra clave para memorizar: medible
Explicación: Resultado concreto, específico y medible que ayuda a cumplir un objetivo.
SMART
Palabra clave para memorizar: claridad
Explicación: Criterio según el cual una meta debe ser específica, medible, alcanzable, relevante y con plazo definido.
Esfuerzo tecnológico
Palabra clave para memorizar: trabajo
Explicación: Acciones y recursos tecnológicos que la organización dedica para lograr algo.
Propósito
Palabra clave para memorizar: sentido
Explicación: Intención fundamental que da sentido a la existencia de la organización.
Valores
Palabra clave para memorizar: principios
Explicación: Ideas o principios que guían el comportamiento y las decisiones de la organización.
Políticas
Palabra clave para memorizar: reglas
Explicación: Lineamientos generales que orientan la forma de actuar y decidir.
4. Análisis de la organización
Para decidir bien en TI no basta con ver solo una parte. Hay que revisar lo de afuera, como mercado, leyes o tecnología, y lo de adentro, como personal, procesos, dinero e infraestructura. Juntas, esas dos miradas dan una visión más completa.
Análisis externo
Palabra clave para memorizar: afuera
Explicación: Revisión de oportunidades y amenazas fuera de la organización.
Análisis interno
Palabra clave para memorizar: adentro
Explicación: Revisión de fortalezas y debilidades dentro de la organización.
PESTEL
Palabra clave para memorizar: entorno
Explicación: Herramienta para analizar factores políticos, económicos, sociales, tecnológicos, ecológicos y legales.
Posicionar la TI estratégicamente
Palabra clave para memorizar: relevancia
Explicación: Hacer que TI sea importante para lograr resultados de la organización.
Apalancar la TI
Palabra clave para memorizar: impulso
Explicación: Usar la tecnología como apoyo para mejorar procesos o resultados.
Visión integral
Palabra clave para memorizar: completo
Explicación: Comprender una situación viendo de forma conjunta lo de afuera y lo de adentro.
Organización como sistema
Palabra clave para memorizar: interrelación
Explicación: Forma de ver a la organización como partes conectadas que se influyen entre sí.
Estructura organizacional
Palabra clave para memorizar: estructura
Explicación: Forma en que se distribuyen áreas, puestos, autoridad y responsabilidades.
Descripción de puestos
Palabra clave para memorizar: funciones
Explicación: Documento o definición que indica tareas, responsabilidades y requisitos de un cargo.
Procedimientos estratégicos
Palabra clave para memorizar: pasos
Explicación: Pasos formales que orientan la ejecución de decisiones estratégicas.
Información financiera
Palabra clave para memorizar: dinero
Explicación: Datos económicos que ayudan a evaluar la situación y tomar decisiones.
Línea de acción
Palabra clave para memorizar: ruta
Explicación: Curso concreto de acciones para ejecutar una estrategia.
Diseño de estrategias
Palabra clave para memorizar: planeación
Explicación: Definición de cómo se buscará cumplir un objetivo.
Evaluación de estrategias
Palabra clave para memorizar: medición
Explicación: Revisión de si una estrategia funciona y qué debe ajustarse.
5. Cadena de valor y actividades de la empresa
La cadena de valor explica que una empresa crea valor con varias actividades conectadas. Unas están más pegadas al producto o servicio; otras lo apoyan desde atrás. Cuando todas funcionan bien, el cliente percibe más valor y la empresa obtiene mejor resultado.
Cadena de valor
Palabra clave para memorizar: valor
Explicación: Conjunto de actividades con las que una empresa crea valor para el cliente.
Actividades primarias
Palabra clave para memorizar: directo
Explicación: Actividades más ligadas a crear, vender, entregar y dar servicio al producto o servicio.
Actividades de apoyo
Palabra clave para memorizar: soporte
Explicación: Actividades que sostienen y potencian a las actividades primarias.
Logística
Palabra clave para memorizar: flujo
Explicación: Organización de lo que entra, se mueve y sale para que el trabajo funcione.
Logística interna
Palabra clave para memorizar: entrada
Explicación: Recepción, almacenamiento y distribución interna de insumos.
Operaciones
Palabra clave para memorizar: transformación
Explicación: Actividad donde los insumos se convierten en el producto o servicio final.
Logística externa
Palabra clave para memorizar: salida
Explicación: Distribución y entrega del producto o servicio al cliente.
Marketing y ventas
Palabra clave para memorizar: persuasión
Explicación: Actividades para dar a conocer, convencer y vender.
Servicio postventa
Palabra clave para memorizar: seguimiento
Explicación: Atención y soporte que se da después de vender.
Abastecimiento
Palabra clave para memorizar: obtención
Explicación: Proceso de conseguir materiales, herramientas, equipos, servicios y proveedores.
Desarrollo tecnológico
Palabra clave para memorizar: innovación
Explicación: Investigación, desarrollo e innovación de productos, procesos y sistemas.
Margen
Palabra clave para memorizar: beneficio
Explicación: Diferencia entre el valor generado y los costos incurridos.
Cadena de valor de servicio
Palabra clave para memorizar: intangibilidad
Explicación: Adaptación de la cadena de valor a actividades donde predomina el servicio y la experiencia del cliente.
Cadena de valor virtual
Palabra clave para memorizar: digital
Explicación: Generación de valor mediante información, datos y procesos digitales.

ERP

Tablero ejecutivo

HRIS

E-learning

Capital humano

Insumo

Material

I+D en IA

Gestión

7. Flujo de trabajo de una empresa de software
Una empresa de software no solo programa. Primero entiende lo que el cliente necesita, luego organiza el trabajo, construye el sistema, lo publica y finalmente lo mantiene funcionando. Cada etapa usa herramientas y conceptos distintos.
Requerimientos
Palabra clave para memorizar: necesidad
Explicación: Necesidades, funciones o reglas que el cliente o usuario requiere.
Levantamiento con el cliente
Palabra clave para memorizar: descubrir
Explicación: Proceso de hablar con el cliente para entender sus necesidades.
Backlog
Palabra clave para memorizar: pendientes
Explicación: Lista ordenada por prioridad de tareas o funcionalidades pendientes.
Jira / Azure DevOps
Palabra clave para memorizar: organización
Explicación: Herramientas para administrar tareas, avances y trabajo del equipo.
Talleres de descubrimiento
Palabra clave para memorizar: exploración
Explicación: Reuniones para entender bien el problema y descubrir qué se debe construir.
Prototipo
Palabra clave para memorizar: prueba
Explicación: Versión de prueba o maqueta de cómo podría verse o funcionar un sistema.
Sprint
Palabra clave para memorizar: periodo

Prueba unitaria

Prueba de integración

Pair programming

Revisión de arquitectura

Docker

8. Herramientas de diagnóstico
Las herramientas de diagnóstico sirven para entender mejor una situación y decidir con más claridad. Algunas ayudan a consultar expertos, otras a describir factores, otras a medir prioridades y otras a encontrar causas o ver la competencia.
Método Delphi
Palabra clave para memorizar: consenso
Explicación: Método de consulta anónima e iterativa a expertos para llegar a una opinión más sólida.
FODA cualitativo
Palabra clave para memorizar: describir
Explicación: Análisis FODA en el que los factores se describen con palabras y sin números.
FODA cuantitativo
Palabra clave para memorizar: medir
Explicación: Análisis FODA al que se le agregan pesos e impactos numéricos para priorizar.
Peso
Palabra clave para memorizar: importancia
Explicación: Valor que indica qué tan importante es un factor frente a otros.
Impacto
Palabra clave para memorizar: fuerza
Explicación: Valor que indica qué tanto influye un factor en la organización.
Puntaje ponderado
Palabra clave para memorizar: producto
Explicación: Resultado de multiplicar peso por impacto.
Priorización estratégica
Palabra clave para memorizar: prioridad
Explicación: Ordenar qué merece más atención según su importancia e impacto.
Fortaleza
Palabra clave para memorizar: ventaja
Explicación: Aspecto positivo interno que beneficia a la organización.
Debilidad
Palabra clave para memorizar: freno
Explicación: Aspecto negativo interno que limita o complica a la organización.
Oportunidad
Palabra clave para memorizar: aprovechar
Explicación: Factor externo favorable que puede beneficiar a la organización.
Amenaza
Palabra clave para memorizar: riesgo
Explicación: Factor externo desfavorable que puede afectar a la organización.
Ishikawa
Palabra clave para memorizar: causas
Explicación: Diagrama causa-efecto que ayuda a identificar causas posibles de un problema.
Pareto
Palabra clave para memorizar: 80/20
Explicación: Análisis que ayuda a detectar los pocos factores que explican gran parte del problema.
5 fuerzas de Porter
Palabra clave para memorizar: competencia
Explicación: Modelo que analiza rivalidad, nuevos entrantes, sustitutos, poder de proveedores y poder de clientes.
9. Tipo de oferta y estrategia aplicada
No todo lo que ofrece una organización es igual. Puede vender bienes, prestar servicios o incluso construir experiencias. Además, para ejecutar estrategias necesita convertir ideas en acciones concretas, seguir procedimientos y evaluar resultados.
Producto
Palabra clave para memorizar: oferta
Explicación: Aquello que una organización ofrece al mercado, ya sea un bien, un servicio o una experiencia.
Bien
Palabra clave para memorizar: tangible
Explicación: Producto físico que puede tocarse.
Servicio
Palabra clave para memorizar: atención
Explicación: Actividad o prestación intangible que satisface una necesidad.
Experiencia
Palabra clave para memorizar: vivencia
Explicación: Valor percibido por el usuario a partir de lo que vive y siente al interactuar con una organización.
Implementación de estrategias
Palabra clave para memorizar: ejecución
Explicación: Puesta en práctica de las acciones definidas para cumplir una estrategia.
